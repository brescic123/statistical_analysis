# Online Store Data Analysis

Ovaj projekat vrši analizu podataka online trgovine korištenjem Python biblioteke Pandas.
Analiza je rađena na temelju zadatih poslovnih pitanja s ciljem boljeg razumijevanja prodaje,
popularnosti proizvoda i efikasnosti.

## Korištene tehnologije
- Python
- Pandas

## Analiza uključuje
- Izračun prosječne ocjene proizvoda
- Identifikaciju najčešćeg brenda u ponudi
- Analizu najprodavanijeg brenda
- Analizu popularnosti proizvoda po bojama
- Grupisanje i sortiranje podataka
- Izračun efikasnosti prodaje
- Rangiranje i prikaz 5 najefikasnijih proizvoda

## Rezultati
Rezultati analize omogućavaju uvid u ponašanje kupaca, prepoznatljivost brendova
i performanse proizvoda, što može biti korisno za optimizaciju ponude i prodajne strategije.
